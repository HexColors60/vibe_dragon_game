請使用 **Rust + Bevy** 建立一款「3D 風格的恐龍狩獵遊戲」，需求如下：

====================
遊戲核心機制
====================
1. 玩家駕駛一台裝有機槍的皮卡車，在侏羅紀風格的環境中追逐恐龍
2. 使用滑鼠瞄準；滑鼠左鍵點擊／按住可射擊機槍
3. 恐龍具備不同身體部位判定（例如：頭部、心臟等要害部位）
   - 命中要害可造成致命／高倍率傷害
4. 恐龍被擊殺後會倒地（死亡動畫／布娃娃或簡化倒地），玩家獲得分數
5. 皮卡車可用鍵盤 **WASD** 或方向鍵控制移動（前後左右）

====================
視覺呈現需求（3D）
====================
* 背景：侏羅紀原始森林（高大蕨類、蘇鐵、松柏等植被）
* 地形：起伏地表，包含岩石、河流、火山景觀
* 天空：營造遠古氛圍，可出現翼龍飛過
* 恐龍種類：至少 3 種（例如：三角龍、迅猛龍、腕龍）
  - 每種恐龍具有不同移動速度與血量
* 光影效果：模擬陽光穿透樹林（例如：方向光 + 陰影、體積光／簡化光束效果）

====================
遊戲元素
====================
* 皮卡車模型：越野風格，車頂安裝可旋轉機槍（機槍跟隨瞄準方向）
* 準星系統：螢幕中央顯示十字準星
* 血量系統：恐龍顯示血條（可在頭頂 UI 或 HUD 上）
* 命中回饋：子彈命中顯示視覺效果（血液噴濺粒子／貼花）
* 音效提示：射擊聲、恐龍吼叫聲、車輛引擎聲

====================
技術實作建議（Rust + Bevy）
====================
* 引擎：使用 **Bevy**（3D 渲染、輸入、音效、UI、ECS）
* 物理與碰撞：
  - 建議使用 **bevy_rapier3d**（或其他物理外掛）做基礎碰撞/射線檢測
  - 子彈可用「射線命中（raycast）」或「彈道投射 + 碰撞體」
* 恐龍 AI：
  - 平時隨機巡邏／遊走
  - 被攻擊後會加速逃跑（可加入簡單狀態機：Idle / Roam / Flee / Dead）
* 計分系統與時間顯示：
  - HUD 顯示分數、剩餘時間或已用時間
* 響應式設計：
  - 自動適配不同解析度／視窗大小（UI 使用相對布局；相機視角可微調 FOV）

====================
交付物
====================
請提供一個可直接執行的 Rust 專案（Cargo），包含：
- Cargo.toml（依賴：bevy、物理外掛、可能的音效/粒子支援）
- main.rs（或分模組：camera/input/vehicle/dino/weapon/ui/ai 等）
- 必要的資源（或用程式生成的簡化模型/材質，確保不依賴外部下載也能跑）
- 專案執行方式（例如：cargo run）

目標：啟動後即可在 3D 場景中駕車、用滑鼠瞄準射擊、擊殺不同恐龍並計分，並具備基本 HUD / 命中回饋 / 音效。
